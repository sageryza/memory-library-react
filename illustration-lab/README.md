# illustration-lab

Permanent archive of everything made in the journal-illustration / timeline work:
generated drawings and comparison sheets, the audio message clips, HTML galleries
and prototypes, render scripts, prompt files, reference sets, the LoRA training
pipeline (`batch1/`), and logs.

Committed here so nothing lives only in the ephemeral session scratchpad.
See the "Saving work (forever)" rule in the repo root CLAUDE.md.

Key subfolders:
- `orig15/`, `raw15_full/`, `model_compare/`, `taste_render*/`, `new_render/`,
  `march_render/`, `redraw/` — generated illustration tests (gpt-image runs).
- `audio/` + top-level `msg_*.mp3` — the spoken message clips.
- `*.html` — the Illustrate prototype and line-quality galleries.
- `*.mjs` — every render/build script (prompts live in these + `chatgpt_grid_prompt.txt`, `final_prompt.mjs`).
- `batch1/` — the LoRA training-data pipeline (erased/cleaned drawings, PSD layers, train sets).
- `handoff/` — the original journal-timeline handoff package.
- `overnight_log.md` — the banding run log.
